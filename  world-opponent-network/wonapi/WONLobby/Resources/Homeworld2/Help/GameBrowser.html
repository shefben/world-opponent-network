Game Browser

The Game Browser displays games that are available to join and/or in-progress. It is available on several screens including the Lobby Tab, LAN Tab, and Direct Tab. The game browser consists of a multi-column list box; each row of the list box represents a game. The various columns provide summary information about each game. If Show Games in Progress Option is enabled, in-progress games are displayed in a disabled color.

Below the list box on the left side is a summary of the games available. It lists the number of games displayed as well as the total games available (displayed and not displayed) in parentheses. If one or more games is currently being refreshed, the number of servers refreshed out of total is displayed.

Game Browser Buttons

The following buttons are available below the list box:

  • Join Game. Attempts to join the currently selected game. This button is only enabled if there is a game selected, and it allows you to join.

  • Host Game. Hosts a new game on your machine. The Host Game Dialog will be displayed.

  • Refresh List. Refreshes the current lists of games. This may take awhile if the number of games is large. Note that while a Refresh of any games is in progress, this button will display Stop Refresh. Clicking will abort the Refresh of any remaining games.

Game Browser Columns

Each column of the game browser provides information about the game; most are self explanatory. The following columns contain special information:

  • Info. The Info column is the left-most column. It contains a triangle which may be clicked to toggle display of extended game details. It will also contain an icon if the game has any entry restrictions.

  • Skill. The Skill column is the second-left-most column. If a game has been designated for a certain skill level (novice, intermediate, advanced, expert), an appropriate icon will be displayed.

  • Ping. The Ping column is the right-most column. It shows a measurement of the latency between your machine and the game machine. The display is a series of green bars; the more green bars, the lower the latency (and the better the game will play). If a game is not responding, the Ping column will contain the Not Responding icon.

Joining Games and Game Information

You can attempt to join a game by double-clicking it, by right-clicking it and selecting connect, or by selecting it and pressing the Join Game button. Note that some games have entry restrictions that may prevent you from joining. If joining is successful, you will be taken to the Game Tab; if joining fails, an appropriate error will be displayed.

You may click on the triangle in the left column of game to toggle display of extended details. Note that opening extended details may take a few seconds as the game must be queried. Extended details are displayed directly below the game in the Game Browser.

You may right click on a game for a popup menu of commands. The available commands will include: connect, refresh server, show details and refresh list.

Join Game Attempt to join the game.
Ping Game Refresh the game details and ping.
Query Details Refresh the game details and ping.
Show/Hide Details Toggle display of extended game details.

Sorting and Filtering

The list of games may be sorted by any column. Click on the column header name to sort the games in ascending order by that column. Click on the same column again to sort the games in descending order by that column.

You may also filter the game list by column(s). Filters may be applied to any column from a combo box below its title. You may select a pre-defined filter from the drop-down list of the combo box or define your own filter by typing one in the combo box. Filters restrict the game list to only games that match the filter. For instance, if you enter Bob in the game name column filter, only games that begin with Bob will appear. Certain columns support more advanced filters that allow comparisons (<, <=, >, >=). Check the sample filters for each column to see examples of what is supported. To remove filtering for a column, select No Filter from the drop-down list.

You may also add your own custom filters and remove filters (including sample filters) for each column. To add, enter the filter in the combo box and select Add in the drop-down list. To remove a filter, select select the filter from the drop-down list or enter the filter in the combo box, then select Remove in the drop-down list. You may reset all filters to their default list in the Options Tab.
