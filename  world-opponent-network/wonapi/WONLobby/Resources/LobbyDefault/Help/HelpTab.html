Using Help - The Help Tab

The Help Tab provides access to the in-game help system. Here you can review details on using the various portions of the multiplayer matchmaking system as well as getting some details on resolving certain types of problems.

The Help Tab functions much like your favorite Internet Browser. The help text contains links that you may click on to go to other help topics. Below the help text are typical browser buttons:

  • Back Button: Returns you to the previous help topic. Back is only available after clicking a link.
  • Forward Button: Returns you to next help topic. Forward is only available after using Back.
  • Stop Button: Stops the load of the current help topic. Stop is only available while a help topic is loading.
  • Refresh Button: Reloads the current help topic.
  • Home Button: Returns you to the first help page.

Additionally, you may search through the currently displayed help topic using the Find controls, located in the lower right. To search the current help topic, enter a string and select Find Next to search forward or Find Prev to search backward. Searches are case-insensitive, unless the Case Sensitive check box is checked.
