Chat

Chat allows discussions between the players in a given room or game staging area. It is available in several locations of the matchmaking system including the Lobby Tab and multiple locations on the Game Tab.

Chat is made up of several controls: the Chat Window, the Chat Command Box, and the User List.

Chat Window

The Chat Window displays all chat from each user in the room. User name is displayed in a different color than chat text. You may left-click on a user name to display a popup menu of available commands. Note that some of the popup commands create a partial command in the Chat Command Box; you must complete the command before sending it.

Chat Command Box

The Chat Command Box is located below the Chat Window and consists of an Edit box and a Send button. Enter your chat and commands here. Hit Return or press the Send button to send your chat or execute the chat command. Simply typing text is considered normal chat that is sent to all users in the room. Chat commands are proceeded by a slash ('/'); any text that begins with a slash is considered a chat command.

User List

There is a User List associated with each Chat Window; usually located to the right of the chat window. The User List displays a list of all users in the chat room. You may right-click on a user name to display a popup menu of available commands. Note that some of the popup commands will create a partial command in the Chat Command Box; you must complete the command before sending it.

Each user may have an icon to its left showing additional state. Note that moderators and administrators will sort to the top of the User List, followed by any friends, followed by other users in the chat room; moderators, friends, and other users are separated by dividers within the User List.

Chat Commands

There are many chat commands available for use. Any text that begins with a slash in the Chat Command Box is considered a chat command. Unknown or invalid commands will generate an error displayed in the Chat Window. 

Note: Several chat commands are available via entries in popup menus when a user name is clicked on in the Chat Window or the User List. Also note that the whisper and ignore commands are available via buttons at the bottom of the User list.

Note: Only the Emote, Ignore, Reply, and Whisper commands are available in chat when playing via LAN or Direct Connect. All commands are available when playing via Sierra Online.

The following table summarizes the available chat commands:

Chat Command Summary

Command Usage Shortcuts Description
Away /away /afk Toggles (on or off) your "away from keyboard" setting.
Block /block username /b Adds username to your list of blocked users.
Clear /clear /clr
/cls		 Clears all text from the Chat Window.
Emote /emote action /e
/em
/me		 Emotes action as if you had done it ("/e laughs" would generate "Joe laughs").
Help /help /h
/?		 Displays a chat command summary.
Ignore /ignore [username] /i Toggles username on your list of ignored users (Username<\i> is added if not on the list or removed if on the list. If no username is specified, the contents of your ignore list is displayed.)
Locate Locate is only available via clicking on a user in the chat window or the user list. Selecting locate will output the room and/or game the user is currently in on your server.
Reply /reply [text] /r Replies (whispers) to the last user who whispered to you. If no text is specified, a whisper command to the user in question is generated in the Chat Command Box. If text is specified, a whisper is sent to the user in question.
Whisper /whisper username [text] /w
/t
/tell		 Sends text to only user username.
The following commands can only be used by the room/game captain
Ban /ban username Prevents user username from joining your room or game. If user is already in your room/game, they are kicked out.
Mute /mute username Prohibits user username from chatting within your room or game.
Unban /unban username Unbans user username within your room or game. If user is not already banned, then unban has no effect.
Unmute /unmute username Unmutes user username within your room or game. If user is not already muted, then unmute has no effect.
The following commands can only be used by a game captain and only apply to games created with join restrictions.
Invite /invite username Invites user username to join your game. If user has already joined your game, then invite has no effect. Inviting a user allows the user to bypass any join restrictions.
Uninvite /uninvite username Revokes invitation for user username to join your game. If the user has not been invited, has already joined the game, or the game has no join restrictions, then uninvite has no effect.
