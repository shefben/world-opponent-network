Ignore List

Ignoring a user will suppress all chat from that user in the Sierra Online lobbies. You may both Ignore or Block users, which are chat commands that target a user. They can be accessed via all the normal methods for user based commands.  

Ignore affects only the user name; block disregards all user names associated with an installation of the game. Block potentially ignores users you have never met (if a game is being shared), so be very sure you wish to block someone before doing so.

Users who are ignored or blocked have an appropriate Icon displayed next to their name in all User Lists.

You can display the contents of your ignore list using the Ignore chat command without specifying a user. The Block list can not be displayed, so again, be very sure you wish to block a user before doing so.
