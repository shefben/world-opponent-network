Game Entry Restrictions

When a game is created, entry restrictions may be attached by the host and will limit which users can join the game. Games with entry restrictions are marked with an appropriate icon in the Info column of the game browser. There are three types of entry restrictions:

Ask To Join
As the host, you will receive a prompt in your game chat window whenever a user wishes to join your game. You must click on accept to allow entry or reject to immediately deny entry.  As a user, you must Ask to Join and await the reply.

Invite Only
As the host, you may invite users by using the invite chat command or by right-clicking on a user in a user list and selecting invite. Only users invited by the game host may join the game. Games you have been invited to join are marked differently than games you have not been invited to join. 

Password
As the host, you must specify the password when the game is created. Users must supply the correct password into the dialog box before they can join the game.
