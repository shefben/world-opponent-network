Quick Play Tab

The Quick Play tab allows you to start playing a game as quickly as possible via Sierra Online. Simply select your skill level (novice is the default), and click the Play Now button. Any available games will be searched, and you will be joined to an appropriate game (if one exists).

If no appropriate game exists, a dialog will be displayed asking if you wish to host a game. Click Ok to host or Cancel to abort. If an appropriate game is found, you will join and be taken to the Game tab to begin the game.

Note: you must be connected to Internet in order to play games via Sierra Online.
