Game Tab

The Game Options tab allows you to set up the game options and/or chat with other players before the game actually starts. The game host may configure game and player options, as needed. Players may configure their individual options as needed. When everyone is ready, the host may start the game.

The upper portion of the game tab contains the game chat window and the game user list. The lower portion consists of tabs for the game/player Options, as well as additional Chat tabs.

Setting Up The Game

Before a game can be started by the host, it must be set up--this includes the host setting all game and player options, and other players setting their own options (if any). Most options will have default values and will not need to be set directly. Once all options are set, each player, other than the host, must select Ready by clicking the button on the upper portion of the game tab. Once all players are ready, the host may click Start Game (also located on the upper portion of the game tab).

Note that players who are ready will have the Ready Icon displayed next to their names in the game user list. The host's Start Game button will be disabled until all players are ready. Players may also click the Ready button again to mark themselves as unready and thus prevent the host from starting the game.

The host may ban a troublesome player (like one who won't click ready) at any time by selecting the user's name in the game user list and clicking the Ban button (located at the bottom of the game user list). This will eject the user from the game and prevent them from returning.

Any player may leave the game at any time before it is started by the host by selecting the Leave Game button (located on the upper portion of the game tab). The host may dissolve the game at any time before it is started by selecting the Dissolve Game button (also located on the upper portion of the game tab).

Game Options

The Game Options tab is located on the lower portion of the game setup screen. This tab contains options and information that are common to all players. Only the captain has the ability to modify these options.

The Campaign field displays the name of the campaign scenario that the captain has chosen to play. A description of this campaign appears in the center window. The specific goals of the campaign, if any, are listed on the far right. If you are the captain, you may select from the list of available campaigns.

The Game Type field selects whether this is a New Game, or a continuation of a Saved Game. If the captain has selected Saved Game you must have a copy of the saved game file to play.

The Difficulty field controls the strength of the computer opponents.

Player Options

The Player Options tab is located on the lower portion of the game setup screen. This tab contains options that are specific to each player. Each player may only modify his or her own options.

The Username field has three options and can only be modified by the captain. Open means that this slot is available to joining players. Computer means that no player may enter this slot as it is reserved for computer use. Captains may use this option when they wish to limit the number of players that may join. Note: Use caution when modifying these fields. If a player is already occupying a slot that you modify to Computer, that player will be removed from the game!

Each user must select a city and a zodiac before they may start the game; no two players may select the same city. If two players have selected the same city, their city fields will highlight in red and the captain will not be able to start the game. Players may have the same zodiac.

Game Chat and User List

All chat in the game chat window will be delivered to all players in the game (unless whispered). The game user list supports some additional features over a normal user list: the game host has the Captain icon next to username and is always displayed at the top of the list. Each player in the game list (other than the host), displays their ping to the game host to the right of their name.

If you are the game host and have chosen the Ask To Join entry restriction, then a message will appear in your game chat window whenever a user wishes to join your game. You can click on accept or reject in the message to allow the user to join or dismiss them.

Team Chat

The Team Chat tab allows you to chat between members of your team (if any).  All chat on this tab goes only to you and your team members (excluding whispers). Note that there is no separate User list for the Team Chat tab.

Lobby Chat

The Lobby Chat tab allows you to chat with users in your current Lobby/room. All chat on this tab goes to all users in your Lobby room (excluding whispers). You can use this tab to ask other users to come join your game or to just to chat with others as you wait for your game to begin.
