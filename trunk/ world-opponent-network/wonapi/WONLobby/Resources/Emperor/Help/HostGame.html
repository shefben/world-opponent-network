Hosting a Game

You may host (create) a game from screens containing a game browser: the Lobby Tab, the LAN Tab, or the Direct Tab. Click on the Host Game button, located at the lower right of the game browser, and the Host Game Dialog will appear, containing the following fields:

  • Game Name. Specify the name of the game you wish to create. The Create Game button will be disabled until a name is specified.

  • Skill Level. Specify a skill level (if any). The default is Open, which implies any skill level is welcome.

  • Entry Restrictions. The radio buttons below skill level determine what, if any, entry restrictions attached to the game. The default is Open, or no entry restrictions. For games created via the Lobby Tab, all entry restrictions will be available. For games created via the LAN or Direct tabs, only Password entry restriction will be available.

  • Password. This field is only visible is the Password Protected entry restriction is selected. Enter thre password that will be required for entry into the game. If this field is visible, a password must be entered before the Create Game button is available.

  • Create Game Button. Click on this button to create the game and go to the Game Tab. Note that this button will not be enabled unless the other fields are valid.

  • Cancel Button. Click on this button to cancel creating a game.
